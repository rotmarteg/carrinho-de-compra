🛒 Simulador de Carrinho de Compras
Projeto prático desenvolvido para simular a experiência de um e-commerce, focando na manipulação dinâmica de itens e cálculos de valores em tempo real.
🎯 Objetivo do Projeto
O foco principal foi exercitar a integração entre a lógica de programação e a interface do usuário (DOM), garantindo que as interações de compra sejam processadas corretamente pelo sistema.
💡 Funcionalidades Principais
• Seleção de Produtos: Menu dropdown para escolha de itens com preços pré-definidos.
• Cálculo de Subtotais: Multiplicação automática de quantidade pelo valor unitário.
• Soma Total: Acumulador dinâmico que exibe o valor total da compra a cada nova adição.
• Limpeza de Dados: Função para reiniciar o carrinho e zerar os valores acumulados.
🛠️ Tecnologias Utilizadas
• JavaScript: Responsável por toda a lógica de cálculo e atualização da interface.
• HTML5: Estrutura semântica para os elementos de seleção e exibição.
• CSS3: Estilização visual para simular um ambiente de compras moderno.
🎓 Contexto Acadêmico
Projeto realizado como parte dos estudos de Lógica de Programação, consolidando o conhecimento em funções, variáveis globais e concatenação de strings para exibição de dados.
